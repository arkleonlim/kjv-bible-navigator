# KJV Bible Navigator

This project provides a simple **web-based viewer for the King James Version (KJV) Bible** using JSON files for all **66 books** of Scripture.

The site loads each book dynamically and supports:

- chapter navigation
- verse anchors
- direct verse linking
- simple GitHub Pages hosting

Example verse link:

```
https://arkleonlim.github.io/kjv-bible-navigator/book.html?book=John#john-1-1
```

This will load the Gospel of John and scroll directly to **John 1:1**.

Example Markdown link:

```
[John 1:1](https://arkleonlim.github.io/kjv-bible-navigator/book.html?book=John#john-1-1)
```
